# 要件定義書: ロイヤルティポイント付与・利用機能

| 項目 | 内容 |
|------|------|
| ドキュメントID | REQ-001 |
| 関連ADR | ADR-0001 |
| 作成日 | 2026-03-22 |
| ステータス | ドラフト |

---

## 1. 概要

### 1.1 目的

購買継続率を高めるため、ポイントプログラムを導入する。ユーザーが購入するたびにポイントを付与し、次回以降の購入時にそのポイントを割引として利用できる仕組みを提供する。

### 1.2 背景

購入促進施策として、ポイントプログラムの導入が決定された。シンプルなルールに基づく実装により、ユーザーへのわかりやすさと開発コストの低減を両立する。

---

## 2. スコープ

本要件定義書は以下の機能を対象とする。

- ポイントの付与（購入時）
- ポイントの利用（決済時）
- ポイントの有効期限管理

---

## 3. 機能要件

### 3.1 ポイント付与

| ID | 要件 |
|----|------|
| FR-001 | システムは購入完了時に、購入金額の1%に相当するポイントをユーザーに付与しなければならない。 |
| FR-002 | 付与ポイントは小数点以下を切り捨て（または四捨五入）し、整数ポイントとして記録しなければならない。（端数処理ルールは実装時に確定すること） |
| FR-003 | ポイントの有効期限は付与日から1年間（365日）とする。 |
| FR-004 | ポイントの付与履歴（付与日時、付与元の購入ID、付与ポイント数、有効期限）をシステムに記録しなければならない。 |

### 3.2 ポイント利用

| ID | 要件 |
|----|------|
| FR-005 | ユーザーは保有ポイントが100ポイント以上の場合に限り、決済時にポイントを利用することができる。 |
| FR-006 | 1ポイントは1円として決済金額に充当する。 |
| FR-007 | ポイントの利用は決済金額を超えない範囲で行える。（利用可能ポイント上限 = 決済金額） |
| FR-008 | ポイント利用後、残高が100ポイント未満になった場合でも、利用済みポイントはそのまま消費される。 |
| FR-009 | ポイントの利用履歴（利用日時、利用先の購入ID、利用ポイント数）をシステムに記録しなければならない。 |

### 3.3 有効期限管理

| ID | 要件 |
|----|------|
| FR-010 | 有効期限を過ぎたポイントは自動的に失効し、利用できなくなる。 |
| FR-011 | 有効期限切れのポイントは失効として記録し、会計処理のための集計が行えるようにする。 |
| FR-012 | ポイント残高の表示においては、有効期限切れのポイントを除いた有効残高のみを表示する。 |

---

## 4. 非機能要件

| ID | 要件 |
|----|------|
| NFR-001 | ポイント付与・利用処理は購入トランザクションと整合性を保つこと（二重付与・二重利用の防止）。 |
| NFR-002 | ポイント残高・履歴はユーザーがいつでも確認できること。 |
| NFR-003 | ポイント計算の丸め方針を明文化し、システム全体で一貫して適用すること。 |

---

## 5. ビジネスルール

| ID | ルール |
|----|--------|
| BR-001 | 付与率: 購入金額の1% |
| BR-002 | 最低利用ポイント: 100ポイント |
| BR-003 | レート: 1ポイント = 1円 |
| BR-004 | 有効期限: 付与日から1年間 |

---

## 6. 制約・注意事項

- **ポイント原資のコスト管理**: ポイント付与に伴うコストを適切に管理・計上する必要がある。
- **有効期限切れポイントの会計処理**: 失効ポイントの会計上の取り扱いを財務部門と合意の上、処理ルールを定めること。

---

## 7. 検証方法

- E2Eテスト: `npm run test:e2e`
  - ポイント付与フロー（購入 → ポイント付与確認）
  - ポイント利用フロー（100ポイント以上保有 → 決済時適用 → 残高減少確認）
  - 有効期限切れポイントの利用不可確認

---

## 8. 用語定義

| 用語 | 定義 |
|------|------|
| ポイント | 購入金額に応じて付与される仮想通貨単位。1ポイント=1円として利用可能。 |
| 有効残高 | 有効期限内のポイントの合計。 |
| 失効 | 有効期限を過ぎて使用不能となったポイントの状態。 |
| 充当 | ポイントを決済金額の一部として適用すること。 |
