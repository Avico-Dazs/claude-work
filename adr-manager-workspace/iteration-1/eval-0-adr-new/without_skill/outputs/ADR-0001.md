# ADR-0001: Adopt Next.js App Router

## Status
Accepted

## Context
We need to decide on the routing strategy for our Next.js project.
App Router is the newer approach recommended by the Next.js team.

## Decision
We will use the App Router (app directory) instead of Pages Router.

## Consequences
- Better performance with React Server Components
- More complex mental model for developers
- Some third-party libraries may not support App Router yet
