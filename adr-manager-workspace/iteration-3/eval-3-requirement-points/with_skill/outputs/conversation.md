# Conversation Log

## User

新機能の話なんですが、ユーザーが商品を購入したとき、自動的にポイントを付与する仕組みを作りたいです。購入金額の1%をポイントとして付与して、100ポイント以上から1ポイント=1円で使えるようにします。この要件で実装進めましょう。

## Assistant

ポイント付与システムの要件を確認しました。整理すると：

- **付与率**：購入金額の 1%
- **利用条件**：100 ポイント以上から利用可能
- **ポイント価値**：1 ポイント = 1 円

シンプルで分かりやすいルール設計ですね。実装にあたっていくつか確認しておきたい点があります：

1. ポイント計算の基準金額は税込みですか、税抜きですか？
2. ポイントの有効期限は設定しますか？
3. 商品返品・注文キャンセル時のポイント取り扱いはどうしますか？

これらは後続の設計で詳細化できるので、まずは今確定しているビジネスルールで実装を進めましょう。

📝 ADR-0001 を記録しました（Requirement: 購入ポイント付与システムの導入）
