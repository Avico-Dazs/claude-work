### REQ-0001: ロイヤルティポイント付与・利用ルール

| 項目 | 内容 |
|------|------|
| **ADR ID** | ADR-0001 |
| **ステータス** | Accepted |

#### 背景・課題

購入促進施策として、ポイントプログラムを導入することになった。ユーザーの購買継続率を高めるため、購入ごとにポイントを還元し、次回購入時に利用できる仕組みが必要。

#### 要件内容

購入金額の1%をポイントとして付与する。100ポイント以上から利用可能で、1ポイント=1円として決済時に充当できる。ポイントの有効期限は付与日から1年間とする。

#### 検証方法

```
npm run test:e2e
```

E2E テストでポイント付与・利用・有効期限の全フローを検証する。

#### 期待される効果

- 購買継続率の向上が期待できる
- シンプルなルールで実装コストが低い

#### リスク・考慮事項

- ポイント原資のコスト管理が必要
- 有効期限切れポイントの会計処理が発生する
